# English to Hangul

English sentence to naturally-sounding (enhanced) Hangul notation.

## Requirement

 - For `convert`
     - Python 3
     - `eng-to-ipa` Python package
 - For `draw`
     - `matplotlib` Python package
     - `TkAgg` Pyplot backend
     - `UnDotum` font
